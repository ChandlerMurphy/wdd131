Site Name

This site name relates to the renounced table top game: Dungeons and Dragons. It draws attention to the site from a large player populace and will provide tools for their gameplay.

Site Purpose

Dungeons and Dragons requires a bit of math, dice rolling, and creativity. Juggling all of these ideas during gameplay can be a challenge for some players. This site will aid in several of these gameplay aspects providing a smoother experience for all players. It will feature a dice roller and random quest/event generator. It's simple design now will allow for furture additions and develpment.
